**About grampacker**

grampacker is a backpacking gear list, weight tracker, and packing tool.

It is built for a simple workflow: keep a reusable gear inventory, build trip-specific packing lists, see clear weight totals, and check items off as you pack.

The app is intentionally small. It focuses on gear, list building, pack weight, sharing, and packing mode instead of trying to be a full trip planner.

grampacker works in a browser on desktop and mobile. Saved list data remains available offline, and packing checkmarks sync when you reconnect.

grampacker is a personal project. It is free to use, with no ads or data harvesting.
